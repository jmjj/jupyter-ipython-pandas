## The first steps with Jupyter, IPython and Pandas 
The use of Jupyter, IPython and Pandas illustrated with a real world
data set. The emphasis of this work is on those steps of data cleansing 
that are usually omitted in other tutorials.

The content of the notebook can be viewed in [https://github.com/jmjj/jupyter-ipython-pandas/blob/master/First_steps_with_Jupyter_IPython_and_Pandas.ipynb](https://github.com/jmjj/jupyter-ipython-pandas/blob/master/First_steps_with_Jupyter_IPython_and_Pandas.ipynb).
